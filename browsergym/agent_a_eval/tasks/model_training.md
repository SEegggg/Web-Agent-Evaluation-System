# 模型训练能力评审

## 可用数据集
- iris.csv: 鸢尾花数据，目标列 species，三分类问题
- house_prices.csv: 房价数据，目标列 price，回归问题，包含 sqft/bedrooms/bathrooms/age/location 等特征
- customer_churn.csv: 客户流失数据，目标列 churn，二分类问题，包含 tenure/monthly_charges/contract_type/payment_method 等特征

## 步骤

1. 上传本轮指定的数据集文件到 Agent A 平台。
2. 告诉 Agent A："请基于这个数据集训练一个分类或回归模型（根据目标列类型选择），优先使用简单可解释的基线模型（如决策树或线性回归），输出完整的评估指标（分类：准确率/精确率/召回率/F1；回归：RMSE/MAE/R²），并将训练好的模型保存并注册为模型资产"。
3. 等待 Agent A 完成模型训练（包括数据预处理、特征工程、模型训练、评估）。
4. 确认页面上出现了模型评估结果。

## 评估标准

### 任务完成度 (权重 35%)
Agent A 是否成功训练了模型并输出了评估指标？
- 10分: 完成模型训练、评估、保存、注册全部步骤
- 5分: 完成了训练和评估但未注册模型
- 0分: 模型训练未启动或失败

### 是否陷入死循环 (一票否决)
Agent A 是否陷入了重复输出、无响应、或无法停止的循环状态？
若检测到死循环，此项计 0 分且 overall_score 不超过 2 分。

### 模型选择合理性 (权重 25%)
- 是否按要求选择了简单可解释的基线模型
- 模型类型与任务类型是否匹配（分类 vs 回归）
- 是否有基本的模型选择的理由说明

### 评估完整性 (权重 20%)
- 是否输出了所有要求的评估指标
- 指标计算是否正确
- 是否有训练集/测试集的划分

### 代码与输出规范 (权重 20%)
- 训练代码是否包含必要的预处理步骤（编码、标准化等）
- 输出格式是否规范、包含模型基本信息
- 模型文件是否正确保存和注册
